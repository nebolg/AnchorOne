## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Expo CLI (installed automatically)

### Installation

1. Install dependencies:
```bash
cd AnchorOne
npm install
```

2. Start the development server:
```bash
npx expo start
```

3. Run on your device:
- **iOS**: Press `i` or scan QR code with Camera app
- **Android**: Press `a` or scan QR code with Expo Go app
- **Web**: Press `w`
